### install git, make, neovim, python3


##### 直连GitHub

```
git clone https://github.com/formattedd/dotfiles ~/.dotfiles
cd ~/.dotfiles 
make github 
```

##### 国内镜像gitee

```sh
git clone https://gitee.com/formattedd/dotfiles ~/.dotfiles
cd ~/.dotfiles
make gitee 
```
