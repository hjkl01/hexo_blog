## yarn && npm
```sh
yarn config set registry 'https://registry.npm.taobao.org'
yarn config get registry

yarn global add yrm
yrm ls
yrm use taobao
yrm test

npm config set registry https://registry.npm.taobao.org
npm config get registry
```
